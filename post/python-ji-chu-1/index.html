2、Python交互模式：
在命令行模式下键入python即可进入，>>>

可以用 exit() 退出，回到命令行模式
3、在命令行模式下切换到指定文件夹：
&在命令行模式下运行.py文件：
cd 目录

4、注释
以 # 开头的都会被忽略
5、缩进
可以使用4个空格的缩进形式，每一层都是用4个空格的缩进
6、Python是对大小写敏感，不能混用
7、数据类型

  • 整数
  • 浮点数（小数）
  • 字符串 使用 ' 或者 ”括起来，里面需要有特殊字符时，需要在前面加转义字符 \ 来让系统知道 （\n换行符 \t制表符 \表示一个斜杠\）（特殊：r '均为不转义内容'）
  • 布尔值 True/False两种状态 可以进行 or and not运算
  • 空值 None 不表示0 ，是一个特殊的空值
  • 还有列表，字典，还能自定义数据类型...
    8、在一行代码写不完到了第二行的时候，行头的...不是代码，只是提示接着上一行内容
    9、变量
  • 可以是任意数据类型
  • 变量名必须是“大小写英文”，“数字”和“_”的组合，且不能数字开头
  • 同一个变量的数据类型可以改变，可以不断对其进行赋予不同数据类型的值，这种也是Python的特点，变量本身类型不固定的，所以Python是动态语言。而像Java等叫静态语言，变量在定义的时候就要给定一个数据类型，并不可改变。
    10、常量
      常用全部大写的变量名表示常量
    11、两种除法
  • “/” 结果是浮点数，即便是整除，结果也会是浮点数
  • “//”也叫“地板除”，两个整数除法仍是整数，只保留了整数部分
  • “%”求余，可以得到余数
    12、字符编码
  • ASCII 编码 --不兼容中文显示
  • Unicode 编码 --可以兼容中文，但是表示英文字母内存会多一倍
  • UTF-8 编码 --是上面两种的结合，可变长编码

    ※Python 3中的字符串是以Unicode 编码，是以Python原生支持多语言
ord() 获取字符的整数表示
chr() 把编码转换成对应的字符
  • Python中字符串类型是 str ，在内存中以Unicode表示（一个字符对应若干个字节）。要在网络传输或者保存到磁盘，需要把str转换为以字节为单位的 bytes
encode() 把str指定为单个字节的bytes
decode() 把bytes转换回去字符串
len() 计算str中包含多少个字符，如果转换为Bytes了，则计算字节数

※ 要让Python解释器能够按照UTF-8读取程序，需要加两条注释

#!/usr/bin/env python3 Linux/OS X下，win会忽略这条
# -- coding: utf-8 -- 告诉解释器按照UTF-8读取代码

13、格式化

  • 使用 % 加类型，作为替换内容的标记
    >>> 'Hello, %s' % 'world'
'Hello, world'
>>> 'Hi, %s, you have $%d.' % ('Michael', 1000000)
'Hi, Michael, you have $1000000.'
%d 整数
%f 浮点数
%s 字符串
%x 十六进制整数
  • 有些时候，字符串里面的%是一个普通字符怎么办？这个时候就需要转义，用%%来表示一个%：
    >>> 'growth rate: %d %%' % 7
'growth rate: 7 %'
  • 使用format() 方法格式化。它会用传入的参数依次替换字符串内的占位符{0}、{1}……，不过这种方式写起来比%要麻烦得多
    >>> 'Hello, {0}, 成绩提升了 {1:.1f}%'.format('小明', 17.125) 'Hello, 小明, 成绩提升了 17.1%'
